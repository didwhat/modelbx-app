🚨 Emergency Controls
Immediate protective actions. No governance vote required. Use only in genuine emergencies — all actions are logged and publicly visible on-chain.
⚠ These controls bypass governance. Every action is permanently recorded on-chain and visible to all users. Only use in genuine emergencies such as an active exploit or unexpected smart contract behaviour.
🔒 Emergency Supply Cap Lower
Immediately lower the MBusd supply cap to halt new minting. Existing positions are unaffected — users can still repay and withdraw. Use this if you detect unusual minting patterns or a suspected exploit.
Presets: 0 (halt minting)100K250K
⏸ Pause All New Minting
Sets the supply cap equal to the current minted supply — effectively blocking any new MBusd from being minted. Existing positions are unaffected. Equivalent to setting cap = current supply. Reversible.
❄ Freeze Swap Routing
Block all new swap transactions. Existing LP positions and borrows are unaffected. Users can still repay, withdraw, and redeem MBusd. Use if a swap exploit is detected.
